For inclusion in the interactive simulator, 
add instruction: 
#define INTERACTIVE 1
 in files FigaroModel.cpp and FigaroModelTemplate.h produced by Fig2Cpp.py

or 

set INTERACTIVE in the building options. 
In Code::Blocks, menu item Project/build options then tab #defines

After compilation with a C++ compiler, you should obtain an executable such as the one in /bin
